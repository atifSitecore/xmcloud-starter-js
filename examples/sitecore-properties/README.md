# Sitecore Properties Real Estate Starter

This Next.js starter is designed for Sitecore XM Cloud, inspired by Ellington Properties. It demonstrates a brochureware real estate site with editable property data, grid, and card components supporting multiple variants.

## Features
- PropertyGrid: Displays all properties under the Properties Page
- PropertyCard: Displays a single property, supports Featured, Compact, Detailed variants
- Editable in Sitecore Page Editor
- Sample data and images

## Getting Started
1. Install dependencies: `npm install`
2. Run the app: `npm run dev`
3. Connect to Sitecore XM Cloud for dynamic content

## Structure
- `/src/components/PropertyGrid.tsx`
- `/src/components/PropertyCard.tsx`
- `/src/pages/index.tsx`
- `/public/images/` (sample images)

## Customization
- Update property data in Sitecore
- Add/edit variants in PropertyCard
- Style to match your brand
